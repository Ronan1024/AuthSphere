# AuthSphere Admin

AuthSphere 管理后台前端，基于 Vue 3、TypeScript、Vite、Element Plus 初始化。

## 本地开发

```bash
npm install
npm run dev
```

默认开发地址：`http://localhost:5173`。

## 常用命令

```bash
npm run typecheck
npm run build
npm run preview
```

## 目录说明

- `src/main.ts`：应用入口，注册 Pinia、Router、Element Plus。
- `src/router`：前端路由定义。
- `src/layouts`：后台主布局。
- `src/views`：业务页面。
- `src/styles`：全局样式。
